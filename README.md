# hospital-mngmt-app-v2-mad2
Hospital Management System (HMS) web application that allows Admins, Doctors, and Patients to interact with the system based on their roles.
This app corresponds to my submission for the Modern Application Development 2 (MAD2) Project, and it utilizes the following frameworks:
- Flask
- VueJS
- Bootstrap
- SQLiteDB
- Redis
- Celery
